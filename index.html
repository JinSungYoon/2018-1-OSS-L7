Project Name : 3D Computer graphics learning with open gl

Introduce

The Open Graphics Library (English: Open Graphics Library, OpenGL for short) is a two-dimensional and three-dimensional graphics standard API specification created by Silicon Graphics in 1992 and supports cross-application programming between platforms. The API can generate complex 3D scenes from simple geometric shapes using about 250 function calls. OpenGL is currently used in areas such as CAD, virtual reality, information visualization, and flight simulation. It is also widely used in the field of computer games, and divides the computer graphics world with Microsoft's Direct3D. Unlike Direct3D, version updates are slow because the standard is based on discussions and suggestions from several related companies. A typical game developed using OpenGL is Id Software's Quake and Doom 3 series. It is currently managed by the non-profit technology consortium, the Khronos Group.

Our Project Goal

  • First, We will understand the contexts of openGL open source and translate it into the graphics field
  • Second, We will add the contents of the openGl that is not implemented in the conde as code.
  • Third, We will correct any mistakes or misspellings in the content and add graphics theorem if necessary
